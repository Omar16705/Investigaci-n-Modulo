¿Que es un operador MOD?


La función MOD devuelve el resto de una división; Si escribimos en PSEINT… Para calcular el cociente, utilizamos la función TRUNC. Lo que calcula esta función es la parte entera de un número.


Cuándo usar MOD( )


Utilice la función MOD( ) esta función para probar si dos números pueden dividirse de manera exacta o aislar el resto de un cálculo de división. Esta función divide un número entre otro y devuelve el resto.


Que entiendo del Operador MOD


El operador MOD se utiliza para obtener el resto de una division entre dos numeros.
